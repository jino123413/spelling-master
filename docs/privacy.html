개인정보처리방침

최종 수정일: 2026년 2월 6일

1. 개인정보 수집 항목

"맞춤법 달인"은 별도의 개인정보를 수집하지 않습니다. 사용자가 입력하는 모든 데이터는 사용자의 기기에만 저장됩니다.

2. 데이터 저장 방식

항목 저장 위치 용도
퀴즈 진행 기록 기기 로컬 스토리지 레벨별 최고 점수, 등급 기록
레벨 해금 상태 기기 로컬 스토리지 해금된 레벨 정보 유지
오답 기록 기기 로컬 스토리지 오답 노트 제공

3. 제3자 제공

본 서비스는 사용자의 데이터를 제3자에게 제공하지 않습니다. 모든 데이터는 사용자의 기기에서만 처리됩니다.

4. 데이터 보관 기간

데이터는 사용자가 직접 삭제하거나, 앱을 삭제하거나, 브라우저 데이터를 초기화할 때까지 보관됩니다.

5. 사용자 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 브라우저 데이터 초기화를 통한 전체 기록 삭제
  • 앱 삭제를 통한 모든 데이터 삭제

6. 아동의 개인정보

본 서비스는 만 14세 미만 아동의 개인정보를 별도로 수집하지 않습니다.

7. 개인정보처리방침 변경

본 방침은 관련 법령 및 서비스 변경에 따라 수정될 수 있으며, 변경 시 앱 내에서 공지합니다.

8. 문의

개인정보 관련 문의사항은 앱 내 피드백 기능을 통해 접수하실 수 있습니다.
